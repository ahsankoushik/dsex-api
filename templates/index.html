API for Dhaka Stock Exchange(dsex)

A prototype API for Dhaka Stock Exchange.
This API gets the date form DSEX
Then does web scraping with help of python and serves the data usign flask on heroku platform.
Note: This api has a certain ammount of delay.

If anyone want to collab on this of suggest something Feel free to send a mail or vist this link

To see the api go to API
You can searchusing "ID" and "TRADING CODE".

examples:
https://dsex-api.herokuapp.com/stocks?ID=222
https://dsex-api.herokuapp.com/stocks?Code=ZEALBANGLA
